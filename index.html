CHIMP

a hierarchical planner for mobile robots.

Download this project as a .zip file Download this project as a tar.gz file

CHIMP is a hierarchical hybrid planner that aims at enabling a mobile robot to use different forms of knowledge in its planning process. It combines HTN-planning and meta-constraint reasoning. It is based on the Meta-CSP Framwork.

Instructions and examples how CHIMP can be used will follow soon!
